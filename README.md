# TailorTalk Assistant 🧵🤖

TailorTalk is a conversational AI assistant built using:
- 💬 Streamlit (Frontend)
- ⚙️ FastAPI (Backend)
- 🧠 Gemini LLM
- 📅 Google Calendar API

## Features
✅ Book appointments  
✅ Check available slots  
✅ Natural language chat interface
